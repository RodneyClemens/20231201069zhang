# 百度贴吧前端设计方案

## 1. 信息架构

### 1.1 核心页面结构

**首页**
- 顶部导航栏：Logo、搜索框、用户信息/登录入口
- 左侧导航：分类导航、我的关注贴吧
- 主内容区：推荐贴吧、热门帖子列表（瀑布流/列表两种展示模式）
- 右侧边栏：热门话题、活跃用户
- 底部：版权信息、友情链接

**贴吧列表页**
- 顶部导航栏（固定）
- 分类筛选器：按主题、热度、最新等筛选
- 主内容区：贴吧列表（卡片式展示），包含贴吧名称、简介、关注数、帖子数
- 分页控件

**贴吧详情页**
- 贴吧头部：贴吧名称、简介、关注按钮、发帖按钮
- 帖子筛选：最新、热门、精华等筛选
- 主内容区：帖子列表（标题、作者、发布时间、回复数、浏览量）
- 分页控件

**帖子详情页**
- 帖子头部：标题、作者信息、发布时间、点赞/收藏/分享按钮
- 帖子内容：富文本内容展示，支持图片、视频嵌入
- 评论区：评论列表、评论输入框
- 相关推荐：相关帖子推荐

**用户个人中心**
- 个人信息区：头像、用户名、简介、关注/粉丝数
- 导航切换：我的帖子、我的评论、我的收藏、设置
- 内容列表：根据导航切换展示对应内容

**发帖页面**
- 所属贴吧选择器
- 标题输入框
- 富文本编辑器：支持文本格式化、图片上传、表情
- 标签输入
- 发布/保存草稿按钮

**登录/注册页**
- 登录表单：用户名/邮箱、密码、记住密码、忘记密码
- 注册表单：用户名、密码、确认密码、邮箱
- 第三方登录入口（预留）

### 1.2 导航结构

**全局导航**
- 顶部固定导航栏
- 移动端底部Tab导航

**局部导航**
- 页面内部筛选/排序控件
- 面包屑导航（多级页面）

**交互导航**
- 模态框（登录、注册、编辑资料）
- 抽屉式侧边栏（移动端菜单）
- 悬浮操作按钮（快速发帖）

## 2. UI 风格说明

### 2.1 设计语言

**整体风格**：简洁现代、专业清晰，注重信息层级和用户体验

**色彩系统**：
- 主色调：蓝色系 #1E88E5（代表信任、专业）
- 辅助色：
  - 深蓝色 #1565C0（强调、重要操作）
  - 浅蓝色 #64B5F6（次要按钮、提示）
  - 绿色 #4CAF50（成功状态）
  - 红色 #F44336（错误状态、警告）
  - 橙色 #FF9800（提示、活动）
- 中性色：
  - 白色 #FFFFFF（背景）
  - 浅灰 #F5F5F5（卡片背景）
  - 中灰 #E0E0E0（分割线、边框）
  - 深灰 #757575（次要文字）
  - 黑色 #212121（主要文字）

**字体系统**：
- 主要字体："PingFang SC", "Microsoft YaHei", "Hiragino Sans GB", sans-serif
- 标题字体大小：
  - H1: 24px, 字重600
  - H2: 20px, 字重600
  - H3: 18px, 字重500
- 正文字体大小：14px-16px, 字重400
- 辅助文字：12px, 字重400

**间距系统**：
- 基础间距单位：4px
- 常用间距：8px, 12px, 16px, 20px, 24px, 32px
- 页面布局间距：
  - 卡片间距：16px
  - 区域间距：24px
  - 内容内边距：16px

### 2.2 视觉元素

**圆角**：
- 按钮：4px-8px
- 卡片：8px
- 输入框：4px
- 头像：50%（圆形）

**阴影**：
- 卡片阴影：0 2px 8px rgba(0, 0, 0, 0.08)
- 悬停阴影：0 4px 12px rgba(0, 0, 0, 0.12)
- 模态框阴影：0 8px 24px rgba(0, 0, 0, 0.16)

**图标系统**：
- 使用线性图标，风格统一
- 尺寸：16px（正文）、24px（标题）、32px（大按钮）
- 颜色：主色、灰色系列

### 2.3 响应式设计

**断点设置**：
- 移动端：< 768px
- 平板：768px - 1024px
- 桌面端：> 1024px

**布局调整**：
- 移动端：单列布局，侧边栏转为抽屉式
- 平板：双列布局，适当隐藏次要信息
- 桌面端：三列布局，完整展示所有内容

**组件适配**：
- 按钮大小：移动端40px高度，桌面端36px高度
- 输入框：移动端宽度100%，桌面端固定宽度
- 字体大小：移动端相对减小

## 4. 切图与资源说明

### 4.1 图标资源

**图标类型**：
- 线性图标（主要功能图标）
- 填充图标（状态指示图标）

**图标尺寸**：
- 导航图标：24px
- 功能图标：16px
- 小图标：12px

**图标命名规范**：
- 格式：`icon-功能名.svg`
- 示例：`icon-home.svg`, `icon-search.svg`, `icon-like.svg`

**图标资源清单**：
1. 导航类：首页、发现、消息、我的
2. 功能类：搜索、发帖、点赞、评论、收藏、分享
3. 状态类：加载中、成功、失败、警告
4. 操作类：编辑、删除、回复、设置

### 4.2 图片资源

**图片类型**：
- Logo图片
- 默认头像
- 示例图片
- 装饰性图片

**图片格式**：
- 图标：SVG（矢量图）
- 照片：JPG/PNG（根据需要选择）
- 小图标/按钮：PNG（带透明度）

**图片命名规范**：
- 格式：`模块名-功能描述-尺寸.格式`
- 示例：`logo-120x40.svg`, `avatar-default-48x48.png`

**图片尺寸规范**：
- Logo：120x40px, 240x80px（高分辨率）
- 头像：40x40px, 80x80px, 120x120px
- 帖子缩略图：80x80px, 160x160px
- Banner图片：1200x300px

### 4.3 字体资源

**主要字体**：
- 中文字体：PingFang SC, Microsoft YaHei, Hiragino Sans GB
- 备选字体：sans-serif

**字体加载策略**：
- 优先使用系统字体
- 可选：CDN加载Google Fonts或其他在线字体

### 4.4 资源目录结构

```
/assets/
  /icons/          # SVG图标
    /nav/          # 导航图标
    /function/     # 功能图标
    /status/       # 状态图标
  /images/         # 图片资源
    /logos/        # Logo图片
    /avatars/      # 头像图片
    /posts/        # 帖子相关图片
    /decorations/  # 装饰性图片
  /fonts/          # 字体文件（如需自定义字体）
  /styles/         # CSS样式文件
    /components/   # 组件样式
    /pages/        # 页面样式
    /variables.css # CSS变量定义
```

### 4.5 响应式图片策略

**图片处理**：
- 使用srcset提供多尺寸图片
- 使用picture元素为不同设备提供不同裁剪版本
- 图片懒加载实现

**示例代码**：
```html
<picture>
  <source media="(max-width: 768px)" srcset="image-mobile.jpg">
  <source media="(max-width: 1024px)" srcset="image-tablet.jpg">
  <img src="image-desktop.jpg" alt="描述" loading="lazy">
</picture>
```

### 4.6 性能优化建议

**图片优化**：
- 压缩图片文件大小
- 使用WebP格式（浏览器支持的情况下）
- 图片CDN加速
- 适当使用CSS渐变替代简单装饰图片

**资源加载优化**：
- 资源预加载
- 关键资源优先加载
- 按需加载非关键资源
- 合理设置缓存策略
