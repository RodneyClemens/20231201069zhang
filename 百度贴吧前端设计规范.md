# 百度贴吧前端设计规范

## 需求理解

目标用户：以年轻群体为主的互联网用户，包括学生、职场人士等，对社区交流、兴趣分享有需求的用户。

使用场景：用户浏览热门帖子、参与话题讨论、发布个人内容、关注兴趣贴吧，主要在碎片化时间使用。

终端支持：Web端桌面应用、移动H5页面，采用响应式设计确保多设备兼容。

风格倾向：简洁现代、清晰易用，注重信息层级和用户体验，以蓝色为主色调，营造信任感和专业感。

## 信息架构 / 页面分区

### 通用布局结构

**顶部区域**
- Logo区域：品牌标识、网站名称
- 搜索框：全局搜索功能
- 用户操作区：发帖按钮、用户头像/登录入口
- 移动端菜单按钮

**左侧区域**（桌面端）
- 主要导航：首页、分类、发现等
- 我的关注：用户关注的贴吧列表
- 快速入口：常用功能快捷访问

**主内容区域**
- 内容筛选/排序工具栏
- 数据展示区：帖子列表、贴吧列表等
- 分页/加载更多控件

**右侧区域**（桌面端，宽度≥1200px时显示）
- 热门推荐：热门话题、热门用户
- 广告位：推广内容
- 社区公告：重要通知

**底部区域**
- 网站导航：关于我们、联系方式等
- 版权信息：版权声明、备案信息
- 友情链接：合作网站链接

**移动端底部导航**
- 首页、发现、发帖、消息、我的

## 组件清单

### 基础组件

**按钮**
- 主要按钮：蓝色背景，用于关键操作
- 次要按钮：灰色背景，用于次要操作
- 文字按钮：仅文字样式，用于非重点操作
- 图标按钮：仅图标样式，节省空间
- 状态：默认、悬停、点击、禁用

**输入框**
- 文本输入框：单行文本输入
- 搜索框：带搜索图标的文本输入
- 多行文本框：用于评论、发帖等
- 状态：默认、聚焦、错误、禁用

**标签**
- 普通标签：显示分类、话题
- 热门标签：醒目样式突出热门内容
- 状态标签：表示内容状态（如：精华、置顶）

**头像**
- 圆形头像：用户头像展示
- 尺寸：小(24px)、中(36px)、大(48px)、超大(64px)
- 状态：默认头像、用户头像

### 内容组件

**帖子卡片**
- 简洁模式：仅显示标题、作者、统计信息
- 详细模式：显示标题、摘要、图片、作者、统计信息
- 状态：默认、悬停、已读

**评论组件**
- 评论输入框：支持多行输入
- 评论列表：显示评论内容、作者、时间
- 回复功能：支持评论回复
- 状态：默认、折叠、展开

**列表组件**
- 表格列表：用于数据管理页面
- 卡片列表：用于帖子、贴吧展示
- 瀑布流：用于图片为主的内容展示

**分页组件**
- 数字分页：显示页码、上一页、下一页
- 加载更多：无限滚动加载
- 状态：当前页、可选页、禁用页

### 交互组件

**导航栏**
- 顶部导航：固定在页面顶部
- 底部导航：移动端固定在底部
- 侧边导航：用于页面内导航
- 状态：默认、滚动时

**下拉菜单**
- 用户菜单：显示用户相关操作
- 功能菜单：显示功能列表
- 状态：默认、展开、悬停

**模态框**
- 登录/注册框：用户认证
- 编辑框：编辑内容
- 确认框：操作确认
- 状态：显示、隐藏

**抽屉**
- 侧边抽屉：移动端导航菜单
- 底部抽屉：操作选项菜单
- 状态：滑入、滑出

**提示组件**
- 成功提示：操作成功反馈
- 错误提示：操作失败反馈
- 警告提示：注意事项提醒
- 状态：显示、隐藏、自动消失

## 视觉基准

### 颜色系统

**主色调**
- 主色：`#007AFF` - 用于主要按钮、链接、高亮状态
- 主色悬停：`#0056b3` - 用于交互元素悬停状态
- 主色禁用：`#B3D4FF` - 用于禁用状态

**辅助色**
- 次要色：`#5856D6` - 用于次要强调、特殊标签
- 成功色：`#34C759` - 用于成功状态、在线状态
- 警告色：`#FF9500` - 用于警告状态、通知
- 错误色：`#FF3B30` - 用于错误状态、删除操作

**中性色**
- 文本主色：`#000000` - 主要文字
- 文本次色：`#8E8E93` - 次要文字、提示信息
- 文本浅色：`#C7C7CC` - 辅助文字、占位符
- 背景主色：`#FFFFFF` - 卡片、弹窗背景
- 背景次色：`#F2F2F7` - 页面背景
- 背景深色：`#E5E5EA` - 输入框背景、分割线
- 边框色：`#C6C6C8` - 边框、分割线

### 字体系统

**字体家族**
- 中文字体：`-apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Helvetica Neue', sans-serif`

**字体大小**
- 大标题：`clamp(1.5rem, 3vw, 2rem)` (24px-32px)
- 中标题：`clamp(1.25rem, 2vw, 1.5rem)` (20px-24px)
- 小标题：`clamp(1.125rem, 1.5vw, 1.25rem)` (18px-20px)
- 正文：`clamp(1rem, 1vw, 1.125rem)` (16px-18px)
- 小正文：`clamp(0.875rem, 0.8vw, 1rem)` (14px-16px)
- 辅助文字：`clamp(0.75rem, 0.7vw, 0.875rem)` (12px-14px)

**字体粗细**
- 粗体：700 (bold)
- 半粗体：600 (semibold)
- 中等：500 (medium)
- 常规：400 (regular)

**行高**
- 标题行高：1.3-1.4
- 正文行高：1.5-1.6
- 紧凑行高：1.2-1.3

### 间距系统

**基础间距单位**
- 间距单位：4px

**容器内边距**
- 页面容器：`clamp(1rem, 4vw, 2rem)` (16px-32px)
- 卡片容器：`clamp(1rem, 2vw, 1.5rem)` (16px-24px)
- 组件容器：`clamp(0.75rem, 1.5vw, 1rem)` (12px-16px)

**区块间距**
- 大间距：`clamp(1.5rem, 3vw, 2.5rem)` (24px-40px) - 主要区域间隔
- 中间距：`clamp(1rem, 2vw, 1.5rem)` (16px-24px) - 卡片间隔
- 小间距：`clamp(0.75rem, 1.5vw, 1rem)` (12px-16px) - 组件间隔
- 微间距：`clamp(0.5rem, 1vw, 0.75rem)` (8px-12px) - 元素内部间隔

**圆角规范**
- 小圆角：4px - 按钮、输入框
- 中圆角：8px - 卡片、弹窗
- 大圆角：12px - 特殊容器
- 全圆角：9999px - 圆形按钮、标签

### 深色模式方案

为了提升用户体验，可考虑实现深色模式作为替代方案。深色模式不仅在夜间使用更舒适，也能减少电池消耗（在OLED屏幕上）。

**深色模式主要颜色调整**：

```css
/* 深色模式颜色变量 */
.dark {
  --primary-color: #3498db;
  --primary-hover: #2980b9;
  --text-primary: #ffffff;
  --text-secondary: #b0b0b0;
  --text-tertiary: #808080;
  --bg-primary: #1a1a1a;
  --bg-secondary: #252525;
  --bg-tertiary: #333333;
  --border-color: #404040;
}
```

**深色模式实现方式**：

1. 使用CSS变量和媒体查询自动检测系统主题
2. 提供手动切换按钮让用户选择偏好
3. 使用localStorage保存用户偏好设置

```jsx
// 深色模式切换组件示例
import React, { useEffect, useState } from 'react';

const ThemeToggle = () => {
  const [isDark, setIsDark] = useState(false);

  useEffect(() => {
    // 检查系统偏好
    const prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches;
    // 检查本地存储
    const savedTheme = localStorage.getItem('theme');
    
    if (savedTheme === 'dark' || (!savedTheme && prefersDark)) {
      document.documentElement.classList.add('dark');
      setIsDark(true);
    }
  }, []);

  const toggleTheme = () => {
    const newTheme = isDark ? 'light' : 'dark';
    document.documentElement.classList.toggle('dark');
    setIsDark(!isDark);
    localStorage.setItem('theme', newTheme);
  };

  return (
    <button 
      onClick={toggleTheme}
      className="p-2 rounded-full hover:bg-bg-secondary transition-custom"
      aria-label={isDark ? '切换到浅色模式' : '切换到深色模式'}
    >
      {isDark ? (
        <i className="fa fa-sun-o text-yellow-400"></i>
      ) : (
        <i className="fa fa-moon-o text-gray-600"></i>
      )}
    </button>
  );
};

export default ThemeToggle;
```

深色模式可以在保持现有布局和功能不变的情况下，通过调整颜色变量提供更加舒适的夜间使用体验，适合长时间浏览社区内容的用户。
